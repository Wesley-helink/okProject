🎯 如何使用評估報告
═══════════════════════════════════════════════════════════════════════

步驟 1: 下載主要檔案
   → 開啟 HMA_開發人天評估表.xlsx

步驟 2: 審查評估內容
   → 檢視「開發人天評估」工作表中的所有功能
   → 確認複雜度和人天估算是否合理

步驟 3: 查看專案摘要
   → 切換到「專案摘要」工作表
   → 檢視總體數據和工期預估

步驟 4: 調整評估 (如需要)
   → 編輯 analyze_hma_estimate.py
   → 修改時薪、複雜度人天、工作項目人天
   → 執行: python analyze_hma_estimate.py
   → 產生新的評估表

步驟 5: 專案規劃
   → 根據評估結果規劃專案時程
   → 確定團隊規模和資源配置
   → 準備專案提案和報價
