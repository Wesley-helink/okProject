## 🔄 如何調整評估

1. 編輯 `analyze_hma_estimate.py`
2. 修改以下參數:
   ```python
   HOURLY_RATE = 1200  # 調整時薪
   COMPLEXITY_ESTIMATES = {
       '簡單': 2,      # 調整各複雜度人天
       '中等': 5,
       '複雜': 10,
       '非常複雜': 15
   }
   ADDITIONAL_ITEMS = [...]  # 調整工作項目人天
   ```
3. 執行: `python analyze_hma_estimate.py`
4. 產生新的 `HMA_開發人天評估表.xlsx`
